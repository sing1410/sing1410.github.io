1). Strong tag

Strong tag makes the text bold and it is used to give importance to the text. Screen readers change their voice for this text.


2). img tag

Img tag is a image tag which is use to insert pictures on a HTML webpage.It is a singular tag and have two attributes alt and src.


3). a tag

Anchor tag is used to link the webpage with other websites, specific pages, photos, documents or any other resource available on the web. For providing the location where the anchor tag will link-to we use the href attribute.


4). ul tag

An unordered list (ul) is used when the items inside the list do not have a specific order or the order is not important. The unordered list starts with an opening tag and ends with a closing tag Inside the opening and closing tags we add each list item (li) in its own opening list item and closing list item tags.


5). form tag

Form tag is used to make forms within the webpage. Form is a paired tag and any information we want to collect using this form will be added inside the opening and closing form tags.
